# colour-detection-system
Colour Detection System is a Python-based project using OpenCV and NumPy that identifies and displays colours in images by analyzing pixel values in RGB/HSV colour space. The system allows interactive colour detection through mouse input, making it useful for basic computer vision learning and image processing applications.

## Features
- Detects colours in images
- Displays RGB and HSV values
- Interactive mouse-based colour selection
- Uses OpenCV for image processing

## Technologies Used
- Python
- OpenCV
- NumPy
